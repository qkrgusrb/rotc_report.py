🇰🇷 ROTC 보고서 자동 생성기 (ROTC Report Generator)
본 프로젝트는 ROTC 후보생의 복잡한 보고 예절과 엄격한 가나다순 정렬 규칙을 준수하여, 누구나 쉽고 정확하게 보고 텍스트를 생성할 수 있도록 설계된 파이썬 프로그램입니다.

🚀 주요 기능 (Key Features)
1. 관등성명 및 보고 양식 자동화
보고 대상(선배, 교관)에 따른 관등성명 및 필수 문구(충성!!, 이상입니다.)를 자동으로 배치합니다.

지침에 명시된 '관등 후 줄바꿈' 및 '마침표 사용' 규칙을 100% 준수합니다.

2. 이중 가나다순 정렬 엔진
참석자 명단: 입력 순서와 상관없이 자동으로 가나다순 정렬을 수행합니다.

열외 사유: 열외 사유 항목 자체를 가나다순으로 정렬한 뒤, 각 사유 내 후보생 이름들까지 다시 가나다순으로 정렬하는 이중 로직을 적용했습니다.

3. 모바일(iPhone/Android) 환경 최적화 (Updated!)
어디서나 보고: PC가 없는 야외 훈련 상황에서도 Pydroid 3(Android)나 Pythonista 3(iOS) 앱을 통해 즉시 실행 가능합니다.

단축어 연동: iOS 단축어(Shortcuts)와 연동하여 홈 화면의 아이콘 하나로 보고 작성을 시작할 수 있습니다.

4. 로컬 기록 관리 및 파일 저장
메모장(.txt) 저장: 생성된 보고서를 스마트폰 내부에 날짜별로 자동 저장하여 과거 보고 이력을 관리할 수 있습니다.

인코딩 최적화: utf-8 설정을 통해 한글 깨짐 없이 안전하게 데이터를 보관합니다.

🛠 사용 방법 (Quick Start)
본 저장소의 rotc.py 코드를 복사합니다.

모바일 파이썬 앱(Pythonista, Pydroid 3 등)에 붙여넣습니다.

프로그램의 안내에 따라 참석자 이름, 열외자 및 사유를 입력합니다.

자동으로 생성된 텍스트를 복사하여 카카오톡에 전송합니다.
